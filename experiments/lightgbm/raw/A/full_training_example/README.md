# Example Experiment: LightGBM Matrix A Full Training

This directory contains an example of a complete experiment output from the AgriFrost-AI training pipeline.

## Directory Structure

```
full_training_example/
├── data_run_metadata.json          # Data processing metadata
├── experiment.log                   # Main experiment log
├── summary.json                     # Overall experiment summary
├── horizon_3h/                      # 3-hour forecast horizon results
│   ├── frost_classifier/           # Frost classification model
│   │   ├── config.json             # Model configuration
│   │   └── model.pkl               # Trained model file
│   ├── temp_regressor/             # Temperature regression model
│   │   ├── config.json
│   │   └── model.pkl
│   ├── frost_metrics.json          # Classification metrics
│   ├── temp_metrics.json           # Regression metrics
│   ├── frost_feature_importance.csv
│   ├── temp_feature_importance.csv
│   ├── reliability_diagram.png     # Probability calibration plot
│   ├── training.log                # Training log
│   └── run_metadata.json
├── horizon_6h/                     # 6-hour forecast horizon
├── horizon_12h/                    # 12-hour forecast horizon
├── horizon_24h/                    # 24-hour forecast horizon
└── loso/                           # Leave-One-Station-Out evaluation
    ├── station_results.json
    ├── station_metrics.csv
    ├── summary.json
    └── station_*/                  # Per-station LOSO results
```

## What's Included

- **Config files**: Model configurations and metadata
- **Metrics files**: Performance metrics (ROC-AUC, PR-AUC, RMSE, MAE, etc.)
- **Feature importance**: Feature importance rankings
- **Model files**: Trained LightGBM models (*.pkl)
- **Visualizations**: Reliability diagrams
- **Logs**: Training and experiment logs

## What's Excluded (Large Files)

To keep the repository size manageable, the following large files have been excluded:

- `labeled_data.parquet` (~32 MB): Preprocessed training data
- `predictions.json` files (~30 MB each): Full prediction arrays

These files can be regenerated by running the training pipeline.

## Total Size

**After excluding large files: ~12 MB**

This example demonstrates the complete structure of experiment outputs while keeping the repository size reasonable for version control.
